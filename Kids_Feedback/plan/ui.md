## 14. Streamlit UI 설계

### 아이 목록 화면

```
아이 선택

[민준] 7세 / 초급 / 실패에 민감함
[서연] 8세 / 초급 / 박자감 연습 필요
[지호] 7세 / 중급 / 준비물 체크 필요
```

### 아이 프로필 카드

```
이름: 민준
나이: 7세
수준: 초급

외우기용 특징:
파란 줄넘기를 들고 오고, 성공하면 양손을 들고 좋아함

성향:
처음에는 조심스럽지만 칭찬을 받으면 적극적으로 도전함

좋아하는 피드백:
개수보다 시도한 점을 칭찬해주면 좋음

주의할 점:
실패를 반복하면 금방 위축될 수 있음

준비물:
개인 줄넘기를 자주 깜빡함
```

### 화면 배치

```
[아이 프로필 카드]
↓
[오늘 수업 기록 입력]
↓
[피드백 생성]
↓
[생성 결과]
↓
[성장 그래프]
```
